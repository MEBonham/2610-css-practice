Commands

Command What's It Do? Example Usage 
ls
 View contents of working directory. 
ls
Desktop Downloads Documents
bin Pictures Videos
 
cd
 Change working directory. 
cd ..
(Moves to parent directory)		 
mkdir
 Create a new directory within the working directory. 
mkdir New-Project
 
pwd
 Print out current working directory path. 
pwd
/home/mckay
 
man
 Display the manual entry for another command. 
man mkdir
Man: find all matching manual pages
Man: What manual page do you want?
 
apropos
 Search by keyword for a new command to use. 
apropos view
aio (7) - POSIX asynchronous I/O overview
bzless (1) file perusal filter ...
...
 
file
 Show the type of a file. 
file commonheader.tex
TeX document, ASCII text
 
cat
 Display the contents of a file to the command line. 
cat commonheader.tex
\usepackage{graphicx} ...
...
 
less
 Display the contents of a file one screen at a time. 
less commonheader.tex
(Displays whole new screen, exit with 'q')		 
type
 Tell what kind of command another command is. 
type less
less is hashed (/usr/bin/less)
 
which
 Tell where the code for another command is located. 
which less
/usr/bin/less
 
rm
 Delete a file. 
rm dummy.txt
 
rmdir
 Delete a folder. 
rmdir Dummy
 
mv
 Move (or rename) a file. 
mv starting-out-with-c.pdf ~/Documents
